## 🚀 Features

- ✅ Generate **strong passwords** 
- ✅ Use **symbols** for increased security
- ✅ Generate **multiple passwords** in one go
- ✅ **Save to file** automatically
- ✅ Optional **verbose output**
- ✅ Works on **Linux, Windows, MacOS**
